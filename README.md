# 🚀 n8n Automation Vault

This repository contains my custom n8n workflows for utility bots and AI agents.

## 📂 Featured Workflows

### Zippy - Unzip Telegram Bot
* **Purpose:** Instant .zip extraction via Telegram.
* **Key Node:** Compression (Unzip).

### Reflect AI - Reflect AI Telegram Bot
* **Purpose:** Socratic brainstorming agent using Llama 3.2.
* **Key Node:** AI Agent + Postgres Memory.

## 🛡️ Security
All JSON files have been exported with **credentials stripped**. You must create your own Telegram, Ollama, and Postgres credentials within n8n to use these workflows.
